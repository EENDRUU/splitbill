Konfigurasi Firebase

Cara dapat Firebase config:

1
Buka console.firebase.google.com → buat project baru
2
Build → Realtime Database → Create database → pilih region Asia Southeast
3
Tab Rules → ubah read & write ke true
4
Project Settings → Web app → Register → copy apiKey, authDomain, databaseURL, projectId
