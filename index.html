Hubungkan Firebase

Masukkan konfigurasi Firebase Realtime Database untuk mulai. Data akan tersimpan di cloud dan bisa diakses semua anggota via link.

Cara dapat Firebase config:

1
Buka console.firebase.google.com → buat project baru
2
Masuk Build → Realtime Database → Create database → pilih region
3
Di Rules, set read & write ke true sementara
4
Project Settings → Web app → copy config (apiKey, authDomain, databaseURL, projectId)
